# plant-stages-holographic-learning-system
An interactive web-based learning platform designed for Grade 4 students that teaches the sunflower life cycle using 3D GLB models and holographic projection technology.

## Features

- Interactive 3D GLB Viewer
- Hologram Projection Mode
- Educational Descriptions
- Audio Narration (Coming Soon)
- Responsive Design
- Multi-user Access

## Technology Stack

- Python
- Django
- HTML
- CSS
- JavaScript
- Three.js
- Bootstrap
